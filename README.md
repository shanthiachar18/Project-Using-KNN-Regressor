# Prediction Using KNN
 Predicting the Bnglore house price.

# Objective
To build a predictive model for Banglore house price.

# Overview
Used Necessory libries to perform EDA, Plotting and to get data information. Used KNN Regressor to predict whether customer will purchase the Iphone or not.

# Methodology
# 1.Preprocessing and EDA:
Performed EDA to get more informative features like null values, outliers, correlation and so on.

# 2.Model Building:
Used K-NearestNeighborsRegressor model.

# 3. Model Validation:
 * R Square values
 * Mean squared error
 * Mean absolute error
